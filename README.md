# Integrating-Jira-Slack

### **TO THE RUN CODE CORRECTLY PLEASE FOLLOW THE STEPS BELOW:**

1. Open command line on the same folder where you have the .py file.
2. Add python "your python name.py" and then '-m' to enter the message and then "your message" hit enter(note: this message will be the name for the Jira issue)
3. You'll recieve 'ok' alert if the code ran correctly and a default message will be sent to your Slack channel and an issue in Jira project.
